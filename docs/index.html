RecyclerView

Normal Style

Normal style of recycler-view.Each item's height is 150px.

'CSS Counter: x' means this item is the xth item in the scroller container.So the max value of x means there are x items in the scroller container totally.

For example, if the max value of x is 14.That means there are 14 items in scroller container totally.

Specially,the red one represent the last item in the scroller container.


Multi Columns Style

Multi columns style of recycler-view.Each item's height is 150px and width is '33.3%'.


Waterfall

Waterfall implementation.Each item's width is '50%' and height is among 100px ~ 400px.


Horizontal Waterfall

Horizontal Waterfall implementation.Each item's height is 150px and width is random among 0% ~ 100%.
